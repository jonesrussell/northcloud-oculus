# Fonts for live feed panel

Place `FiraSans-Bold.ttf` here for the VR live feed panel text to render.

You can copy it from the [Bevy repo](https://github.com/bevyengine/bevy/blob/main/assets/fonts/FiraSans-Bold.ttf) or use another TTF font and update `setup_feed_panel` in `src/main.rs` to load your font path.
